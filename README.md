My first HTML & CSS project: A simple, clean tribute page dedicated to Jesus Christ.

## Live Demo
🌐 [View the live site](https://abz-v0.github.io/css-tribute-page/)

## What I Built
A responsive single-page tribute with:
- Semantic HTML structure (main, figure, blockquote, ul, etc.)
- Centered layout with max-width container
- Image with caption
- Timeline list of key events
- Inspirational quote with citation
- External link to Wikipedia

## Technologies Used
- HTML5
- CSS3 (Flexbox not used yet – mostly block/inline + centering with margin auto)
- Responsive design basics (media queries added later if needed)

## What I Learned
- Basic semantic HTML tags (<main>, <figure>, <figcaption>, <blockquote>, <cite>)
- Centering elements (margin: 0 auto)
- Image responsiveness (max-width: 100%; height: auto)
- Box model basics (padding, margin, border-radius, box-shadow)
- Hover effects on links
- Deploying to GitHub Pages

## How to Run Locally
1. Clone the repo: `git clone https://github.com/Abz-v0/css-tribute-page.git`
2. Open `index.html` in your browser (or use VS Code + Live Server extension)
